Before you start

시작 전에 준비할 것

02

담당 교수에게 아래 세 가지를 받은 뒤, Jupyter 토큰은 허가된 SSH 접속으로 직접 확인합니다. 이 문서에는 실제 비밀번호나 토큰이 들어 있지 않습니다.

받아야 할 정보용도확인
Tailscale 초대허가된 학생임을 확인하는 네트워크 로그인□
학생 폴더명students/<학생 폴더명>의 개인 기본 작업 위치□
공동 SSH 인증정보최초 Jupyter 토큰 확인과 선택적 터미널 접속□

토큰과 비밀번호는 메신저 공개방·GitHub·과제 파일에 올리지 마세요.
화면 캡처를 보낼 때도 주소창의 token 값이나 로그인 입력값이 보이지 않는지 확인합니다.

Windows

tailscale.com/download/windows

Notebook에 로그인하기

03

1. 브라우저에서 주소 열기

기본 주소
http://100.116.76.42:8891/claude-notebook

Chrome 또는 Edge를 권장합니다. 이 주소는 공개 인터넷 주소가 아니므로 Tailscale이 Connected 상태일 때만 열립니다.

주소가 http://로 시작해 브라우저에 “안전하지 않음”이 표시될 수 있습니다. 접속은 허가된 Tailscale 내부망에서만 가능하도록 제한되어 있습니다. 주소를 임의로 바꾸지 마세요.

2. SSH로 Jupyter 토큰 확인

Windows PowerShell 또는 macOS Terminal에서 아래 명령을 실행합니다.

ssh bai001@100.116.76.42 "cat ~/.jupyter/claude-notebook-token"
  1. 처음 연결할 때 질문이 나오면 주소를 확인하고 yes를 입력합니다.
  2. 담당 교수에게 받은 공동 SSH 비밀번호를 입력합니다. 입력 중 글자가 보이지 않는 것이 정상입니다.
  3. 화면에 출력된 한 줄짜리 토큰을 복사합니다.
출력된 토큰은 Notebook 로그인에만 사용하고 채팅방·과제 파일·화면 캡처에 남기지 않습니다. 토큰을 확인한 SSH 창은 바로 닫아도 됩니다.

3. 브라우저 로그인

  1. 로그인 화면의 Password or token 입력칸에 복사한 토큰을 붙여넣습니다.
  2. Log in을 누릅니다.

4. 내 폴더로 바로 시작하기

담당 교수가 학생 폴더명을 알려줬다면 기본 주소 뒤에 다음 query를 붙인 전용 시작 주소를 사용할 수 있습니다.

http://100.116.76.42:8891/claude-notebook?folder=students/<학생 폴더명>

예시: 폴더명이 student-01이면 마지막 부분은 folder=students/student-01입니다.

정상 접속 기준: 로그인 후 상단에 WORKING FOLDER, >_, Claude 버튼이 보입니다.

SSH 접속과 공동 사용 규칙

05

SSH는 토큰 확인과 필요한 경우에 사용합니다

최초 토큰 확인 외에 파일 전송이나 별도 터미널이 필요할 때 Windows PowerShell 또는 macOS Terminal에서 다음 명령을 실행합니다.

ssh bai001@100.116.76.42
  1. 처음 연결할 때 host authenticity 질문이 나오면 주소가 100.116.76.42인지 확인하고 yes를 입력합니다.
  2. 담당 교수에게 받은 공동 SSH 비밀번호를 입력합니다. 입력 중에는 화면에 글자나 별표가 표시되지 않는 것이 정상입니다.
  3. 브라우저 Notebook을 사용할 때는 이 SSH 창을 계속 열어 둘 필요가 없습니다.
passwd를 실행하지 마세요. 공동 계정 비밀번호를 바꾸면 다른 학생이 모두 접속하지 못합니다. sudo 권한 요청이나 시스템 서비스 변경도 금지합니다.

✓ 해도 되는 일

  • 지정된 학생 폴더에서 수업·연구 작업
  • shared/에서 합의된 공동 편집
  • 필요한 Python·Node 명령의 일시 실행
  • 문제가 생기면 오류와 시간을 기록해 문의

! 하면 안 되는 일

  • 계정 비밀번호·Jupyter 토큰·Claude 인증 변경
  • 다른 내부 장비·허용되지 않은 포트 접근 시도
  • 상시 서버·자동 시작·원격 접속 도구 설치
  • 다른 학생 파일을 합의 없이 삭제·이동

동시 사용 에티켓

  • 필요 없는 Claude·터미널 탭은 닫아 메모리를 돌려줍니다.
  • 대규모 설치·학습·빌드 작업은 시작 전 팀에 알립니다.
  • 같은 파일을 동시에 수정할 때는 담당 구간을 먼저 정합니다.
  • 공동 토큰과 비밀번호를 허가받지 않은 사람에게 전달하지 않습니다.

문제가 생겼을 때

06
증상먼저 확인해결
주소가 열리지 않음Tailscale이 Connected인지, 초대받은 이메일인지계정을 전환하고 Tailscale을 다시 연결한 뒤 새로고침
토큰 로그인이 반복됨앞뒤 공백 없이 토큰 전체를 복사했는지시크릿 창에서 다시 로그인. 토큰을 채팅방에 올리지 말 것
WORKING FOLDER가 안 보임오래 열린 탭이나 캐시 사용 여부Windows Ctrl+F5, macOS Cmd+Shift+R
Claude가 시작되지 않음여러 Claude 탭을 동시에 열었는지불필요한 탭을 닫고 30초 후 한 번만 다시 실행
SSH 비밀번호 입력이 안 보임커서만 있고 글자가 표시되지 않는지정상 동작. 그대로 입력 후 Enter

연결 자체를 확인하는 명령

Windows PowerShell

Test-NetConnection 100.116.76.42 -Port 8891

TcpTestSucceeded : True면 네트워크 연결은 정상입니다.

macOS Terminal

nc -vz 100.116.76.42 8891

succeeded가 나오면 네트워크 연결은 정상입니다.
