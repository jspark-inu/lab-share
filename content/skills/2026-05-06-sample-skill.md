# Sample Skill

이 카테고리는 **Claude Code·에이전트가 직접 돌리는 스킬**을 공유하는 자리입니다. 학생이 직접 만들었거나 외부에서 가져와 검증한 *작동하는* 도구를 모읍니다.

## Resources 와의 차이

- **Skills (여기)** — 에이전트가 *돌리는* 코드/워크플로우. 즉시 실행 가능
- **Resources** — 사람이 *읽는* 외부 학습 자료. 영상·책·논문·슬라이드 등

## 본문에 쓸 것

- **무엇을 하는 스킬인지** — 한 줄 요약
- **언제 쓰면 좋은지** — 적합한 상황과 부적합한 상황
- **어떻게 시작하는지** — 셋업 + 가장 작은 작동 예시
- **함정** — 빠지기 쉬운 오해, API 키 같은 보안 주의

## 새 스킬 추가하는 법

`content/skills/YYYY-MM-DD-이름.md` 로 파일 생성 → 이 글의 frontmatter 복사·수정 → push.
