# Lab Ops Hub 개시 — 학부연구생 환영

> 다시 만들지 말고, 공유하자.

## 개요

이 사이트는 박준성 연구실(인천대 데이터사이언스학과)의 **운영·연구 허브**입니다. 회의록, 프로젝트 진행 상황, 공유 스킬, 막힌 부분 wiki 가 한 곳에 모입니다.

## 5 섹션

- **Projects** — 주제별 연구·작업 트래커. 단기/중기/장기 결과물 + 주간 진행
- **Meetings** — 회의록 (자동 인입: 영상 → md → repo)
- **Skills** — 학생들이 발견·작성한 도구·워크플로우
- **Wiki** — 연구 막힌 부분 누적 + 환경/맥락 노하우
- **Notice** — 공지·일정·마감

## 학부연구생 사용법

1. 모바일에서 그냥 읽으세요. 검색창은 한국어/영어 둘 다 됩니다.
2. 글에 의견이 있으면 페이지 하단 **댓글** 에 작성 (giscus + GitHub 계정).
3. Phase 1 — 글 작성은 Claude Code(교수)가 대신 처리. 학생은 의견·요청을 댓글로.
4. Phase 2 (예정) — 학생이 직접 사이트에서 폼 입력 → 자동으로 PR 생성.

## 운영 원칙

- 글의 SSOT는 GitHub repo (`jspark-inu/lab-share`)의 `content/` 폴더.
- 댓글의 SSOT는 GitHub Discussions.
- 사이트는 push 시 자동 배포됨 (~1분).

## 김도현·이혜진·박경 — 첫 댓글 환영

각자 한 글에 댓글 1개 남겨주세요. 어떤 내용이든 괜찮습니다 — 동작 확인용입니다.
