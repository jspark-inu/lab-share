# HAI 연구실 Claude Notebook 학생 시작 가이드

`hai_2` 공동 작업 환경에 처음 접속하는 학생을 위한 6쪽짜리 시작 안내서입니다. Tailscale 연결부터 Notebook 로그인, 작업 폴더 선택, Claude Code 실행, SSH 접속, 공동 사용 규칙과 문제 해결까지 한 번에 확인할 수 있습니다.

## 누가 봐야 하는가

- HAI 연구실 Claude Notebook을 처음 사용하는 학생
- 새 학생의 접속과 작업 폴더 설정을 안내하는 운영자
- 공동 계정 환경의 보안·사용 규칙을 다시 확인하려는 구성원

## 시작점

아래 첨부 HTML을 열어 2쪽의 준비물부터 순서대로 진행하세요. 브라우저에서 바로 볼 수 있고 인쇄 메뉴를 사용하면 A4 안내서로 저장할 수 있습니다.

## 접속 전 주의

Notebook은 공개 인터넷 서비스가 아니라 허가된 Tailscale 네트워크 안에서만 열립니다. 이 안내서에는 실제 비밀번호나 토큰이 없으며, 접속에 필요한 초대·학생 폴더명·공동 SSH 인증정보는 담당 교수에게 별도로 받아야 합니다.

공동 토큰과 비밀번호, 개인 API 키, 공개하면 안 되는 연구자료는 채팅방·과제 파일·공동 Notebook에 남기지 마세요.
