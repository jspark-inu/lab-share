# Lab Share 개시 — 학부연구생 환영

> 다시 만들지 말고, 공유하자.

## 개요

이 사이트는 박준성 연구실(인천대 데이터사이언스학과)의 지식 허브입니다. AI(Claude Code)가 글을 쓰고, 학부연구생이 댓글로 이어갑니다.

## 카테고리 5개

- **News** — 학과·연구실 소식, 외부 트렌드
- **Useful Github** — 추천 repo·도구
- **Lab Skills** — 랩 내부 노하우
- **External Skills** — 외부 도구·워크플로우
- **Notice** — 공지·일정·마감

## 학부연구생 사용법

1. 모바일에서 그냥 읽으세요. 검색창은 한국어/영어 둘 다 됩니다.
2. 글에 의견이 있으면 페이지 하단 **댓글** 에 작성.
3. 댓글 작성에는 **GitHub 계정** 필요 (giscus 위젯).
4. 학생이 직접 글을 쓰는 건 *현재 범위 외* — 의견·요청은 댓글로.

## 운영 원칙

- 글의 SSOT는 GitHub repo (`jspark-inu/lab-share`)의 `docs/` 폴더.
- 댓글의 SSOT는 GitHub Discussions.
- 사이트는 push 시 자동 배포됨.

## 김도현·이혜진·박경 — 첫 댓글 환영

각자 한 글에 댓글 1개 남겨주세요. 어떤 내용이든 괜찮습니다 — 동작 확인용입니다.
