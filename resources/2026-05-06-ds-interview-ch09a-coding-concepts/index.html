Chapter 9. Coding — 개념

Chapter 9 중 개념 파트. 데이터 직무 코딩 면접에서 알아야 할 자료구조·알고리즘 핵심.

다루는 것

  • 자료구조 빅 픽처: array/list/dict/set/stack/queue/heap
  • 시간복잡도 직관: O(n) vs O(n log n) vs O(n²) 어디서 차이 나나
  • 데이터 직무 코딩 면접의 특이점: 풀스택 SWE 면접과 어떻게 다른가
  • pandas/numpy로 풀어도 되는 문제 vs 순수 파이썬으로 풀어야 하는 문제

누가 봐야 하는가

  • CS 전공 아닌데 데이터 직무 코딩 면접 준비하는 학생
  • 알고리즘 문제 풀어봤는데 데이터 직무용으로 정리 안 된 학생
  • LeetCode 막막한 학생 (어디부터 시작할지)
